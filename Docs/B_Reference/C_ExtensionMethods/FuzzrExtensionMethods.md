# Fuzzr Extension Methods
QuickFuzzr provides a collection of extension methods that enhance the expressiveness and composability of `FuzzrOf<T>`.
These methods act as modifiers, they wrap existing fuzzrs to alter behavior, add constraints,
or chain side-effects without changing the underlying LINQ-based model.
  
## Contents
| Method| Description |
| -| - |
| [ExtFuzzr.Apply(this FuzzrOf&lt;T&gt; fuzzr, Action&lt;T&gt; action)](Methods/ExtFuzzrApply.md)| Executes a side-effect for, or transform each value generated by the wrapped fuzzr. |
| [ExtFuzzr.AsObject(this FuzzrOf&lt;T&gt; fuzzr)](Methods/ExtFuzzrAsObject.md)| Boxes generated values as `object` without modifying them. |
| [ExtFuzzr.Many(this FuzzrOf&lt;T&gt; fuzzr, int number)](Methods/ExtFuzzrMany.md)| Produces a number of values from the wrapped fuzzr. |
| [ExtFuzzr.NeverReturnNull(this FuzzrOf&lt;T?&gt; fuzzr)](Methods/ExtFuzzrNeverReturnNull.md)| Filters out null values, retrying until a non-null value is produced or the retry limit is exceeded. |
| [ExtFuzzr.Nullable(this FuzzrOf&lt;T&gt; fuzzr)](Methods/ExtFuzzrNullable.md)| Converts a non-nullable value-type fuzzr into a nullable one with a default 20% null probability. |
| [ExtFuzzr.NullableRef(this FuzzrOf<T> fuzzr)](Methods/ExtFuzzrNullableRef.md)| Wraps a reference-type fuzzr to sometimes return null (default 20% chance). |
| [ExtFuzzr.Shuffle&lt;T&gt;(this FuzzrOf&lt;IEnumerable&lt;T&gt;&gt; source)](Methods/ExtFuzzrShuffle.md)| Randomly shuffles the sequence produced by the source fuzzr. |
| [ExtFuzzr.Unique&lt;T&gt;(this FuzzrOf&lt;T&gt; fuzzr, object key)](Methods/ExtFuzzrUnique.md)| Ensures all generated values are unique within the given key scope. |
| [ExtFuzzr.Where(this FuzzrOf&lt;T&gt; fuzzr, Func&lt;T,bool&gt; predicate)](Methods/ExtFuzzrWhere.md)| Filters generated values so only values satisfying the predicate are returned. |
| [ExtFuzzr.WithDefault(this FuzzrOf&lt;T&gt; fuzzr, T def = default)](Methods/ExtFuzzrWithDefault.md)| Returns def instead of throwing when the underlying fuzzr fails due to empty choices. |
