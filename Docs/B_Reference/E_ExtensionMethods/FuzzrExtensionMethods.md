# Fuzzr Extension Methods
QuickFuzzr provides a collection of extension methods that enhance the expressiveness and composability of `FuzzrOf<T>`.
These methods act as modifiers, they wrap existing fuzzrs to alter behavior, add constraints,
or chain side-effects without changing the underlying LINQ-based model.
  
## Contents
| Method| Description |
| -| - |
| [Apply](Methods/ExtFuzzrApply.md)| Executes a side-effect for, or transform each value generated by the wrapped fuzzr. |
| [AsObject](Methods/ExtFuzzrAsObject.md)| Boxes generated values as `object` without modifying them. |
| [Many](Methods/ExtFuzzrMany.md)| Produces a number of values from the wrapped fuzzr. |
| [NeverReturnNull](Methods/ExtFuzzrNeverReturnNull.md)| Filters out null values, retrying until a non-null value is produced or the retry limit is exceeded. |
| [Nullable](Methods/ExtFuzzrNullable.md)| Converts a non-nullable value-type fuzzr into a nullable one with a default 20% null probability. |
| [NullableRef](Methods/ExtFuzzrNullableRef.md)| Wraps a reference-type fuzzr to sometimes return null (default 20% chance). |
| [Shuffle](Methods/ExtFuzzrShuffle.md)| Randomly shuffles the sequence produced by the source fuzzr. |
| [Unique](Methods/ExtFuzzrUnique.md)| Ensures all generated values are unique within the given key scope. |
| [Where](Methods/ExtFuzzrWhere.md)| Filters generated values so only values satisfying the predicate are returned. |
| [WithDefault](Methods/ExtFuzzrWithDefault.md)| Returns a default value when the underlying fuzzr fails due to empty choices. |
